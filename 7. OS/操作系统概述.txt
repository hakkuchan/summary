OS的大致定义：OS是管理硬件和软件的程序。对于硬件，OS用于资源（CPU、内存等）管理、外设管理等。对于应用程序，OS管理应用程序、为应用程序提供服务等。（硬件之上，应用程序之上）


OS 分为 kernel 和 shell：
OS kernel：CPU调度器、物理内存管理、虚拟内存管理、文件系统管理、中断处理和设备驱动。
OS shell：图像化界面 或 命令符界面。

* CPU → OS → 进程
* 磁盘 → OS → 文件
* 内存 → OS → 地址空间


OS kernel 的特征：
（1）并发：OS管理和调度同时存在多个运行的程序；
（2）共享：“同时”访问 或 互斥共享。
（3）虚拟：把一台物理机器虚拟成多台机器，为不同用户服务。从用户角度来讲，会感觉在独享一台计算机。
（4）异步：程序走走停停，运行时间不可预知，但只要运行环境相同，运行结果也要相同。

* 并行和并发：
* 并行：在某时间点，同时运行多个程序，单核CPU无法并行。
* 并发：在一段时间内，同时运行多个程序。
