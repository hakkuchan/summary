1. 概述

· PyInstaller 用于把 .py文件 打包为 .exe可执行的程序（Windows系统）
  也支持把 .py文件打包为Mac或Linux系统上的程序
  
· 安装或更新：
  pip install pyinstaller
  pip install --upgrade pyinstaller
  
2. 使用方法

· 假定 E:\Work\Jupyter\EXE 路径下保存有 example.py 文件

· 在当前路径下打开 cmd 命令行

· 输入：pyinstaller -F example.py
  回车确认后，pyinstaller便会生成一系列文件，包括：
  文件夹：__pycache__, build, dist
  文件：example.spec
  
 · 其中文件夹 dist 中包含着 example.exe 文件
   可以删除的：__pycache__, build, example.spec
